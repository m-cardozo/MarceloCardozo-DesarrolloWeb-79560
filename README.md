# Proyecto Final - Desarrollo Web

Este proyecto fue desarrollado como trabajo final para el curso de Desarrollo Web de Coderhouse. Está construido utilizando HTML y SASS, aplicando los conocimientos adquiridos durante el curso en cuanto a estructura, organización del código, estilos y diseño web responsive.

El sitio web tiene un enfoque institucional y está inspirado en la marca Volvo Trucks, destacando su trayectoria, innovación y liderazgo dentro de la industria del transporte.
El sitio cuenta cuenta con diferentes páginas diseñadas para brindar una experiencia de navegación clara e intuitiva:

- Inicio: presenta información general y aspectos destacados de la marca.
- Sobre Nosotros: describe los valores de Volvo con la calidad y la innovación.
- Camiones: muestra distintos modelos de camiones y un breve texto introductorio.
- Noticias: informa sobre novedades relacionadas con la marca y el sector del transporte.
- Contacto: permite a los usuarios comunicarse con asesores especializados mediante un formulario de contacto.
